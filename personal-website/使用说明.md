# 网站使用说明

---

## 一、文件结构

```
personal-website/
├── index.html       首页（自动）
├── about.html       关于页
├── style.css        样式（不需要动）
├── app.js           程序（不需要动）
├── posts.json       ← 你的所有帖子在这里
└── images/          ← 你的所有图片放这里
```

---

## 二、发布新帖子

打开 `posts.json`，在 `"posts": [` 后面**最顶部**添加新条目（新帖在前）：

### 纯文字帖子
```json
{
  "id": "4",
  "date": "2026-06-01",
  "title": "可选标题",
  "content": "第一段内容。\n\n第二段内容（两个\\n代表换段）。",
  "images": [],
  "type": "text"
},
```

### 纯图片帖子
```json
{
  "id": "4",
  "date": "2026-06-01",
  "title": "",
  "content": "",
  "images": ["图片文件名.jpg"],
  "type": "image"
},
```

### 图文混合帖子
```json
{
  "id": "4",
  "date": "2026-06-01",
  "title": "可选标题",
  "content": "这里写文字内容。",
  "images": ["图片文件名.jpg", "第二张图.jpg"],
  "type": "both"
},
```

**注意：**
- `id` 每次 +1，不能重复
- 图片文件名要和 `images/` 文件夹里的文件名完全一致
- 每条记录后面有逗号，最后一条没有逗号

---

## 三、添加图片

把图片文件拖进 `images/` 文件夹，文件名建议用英文或数字，避免中文（如 `photo-001.jpg`）。

---

## 四、修改关于页

打开 `about.html`，找到这段，修改成你自己的介绍：

```html
<p>在这里写关于你自己的介绍。</p>
<p>你可以写你是谁，你在记录什么，或者任何你想说的话。</p>
```

---

## 五、修改网站名称

打开 `index.html` 和 `about.html`，把 `SITE NAME` 改成你想要的名字（建议英文大写）。

同时修改 `<title>` 标签里的文字为你的站名。

---

## 六、上线发布（Netlify）

1. 打开 [netlify.com](https://netlify.com)，注册免费账号
2. 登录后，点击 **"Add new site" → "Deploy manually"**
3. 把整个 `personal-website` 文件夹**拖拽**到页面上
4. 等待几秒，你的网站就上线了，会得到一个免费地址

**每次更新内容后，重复步骤 3 重新拖拽即可。**

---

## 七、本地预览

由于浏览器安全限制，直接双击 `index.html` 可能图片加载不出来。
推荐安装 VS Code + Live Server 插件进行本地预览。
